How to Use
  1. Select the transaction date range from dropdown
  2. Enter UTR with RBISH format and click Search
  3. Search New UTR with BiMS > UTR Search
  4. All UTRs starting with RBISH000 are now available on the BiMS website. If not please search with the format "/XUTR/RBISH000"
